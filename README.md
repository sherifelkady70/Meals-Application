## Meals Application
This Android application delivers meals updates to your fingertips, built with modern technologies.

* **Technologies:**
     * Kotlin.
     * Jetpack Compose.
     * Clean Architecture.
     * MVVM.
     * Repository Pattern.
     * Romm DB.
     * Hilt.
